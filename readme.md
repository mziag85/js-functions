სავარჯიშო სახლში : Default Parameters
საყიდლების სია 
წარმოიდგინე, რომ ვწერთ ფუნქციას makeShoppingList(), რომელიც ადგენს საყიდლების სიას.
 თუმცა, არის პროდუქტები, რომლებსაც ყოველთვის ვყიდულობთ: რძე, პური და კვერცხი.
იმისათვის, რომ ფუნქციის გამოძახებისას ყოველჯერზე ამის წერა არ მოგვიწიოს, გამოვიყენოთ Default Parameters.

ნაბიჯი 1: Default Parameter–ების მინიჭება
შეცვალე makeShoppingList() ფუნქციის პარამეტრები ისე, რომ მათ ჰქონდეთ შემდეგი საწყისი მნიშვნელობები:

item1-ისთვის განსაზღვრე 'milk'
item2-ისთვის განსაზღვრე 'bread'
item3-ისთვის განსაზღვრე 'eggs'

ნაბიჯი 2: ფუნქციის გამოძახება არგუმენტების გარეშე
ახლა გამოიძახე makeShoppingList() ყოველგვარი არგუმენტის გადაცემის გარეშე.
 ნახე, როგორ გამოიყენებს JavaScript შენ მიერ მითითებულ "სარეზერვო" მნიშვნელობებს.

ნაბიჯი 3: მნიშვნელობის გადაფარვა
სცადე ფუნქციის გამოძახება და პირველ არგუმენტად გადაეცი სხვა რამ, მაგალითად 'apples'. 

დააკვირდი, რომ 'apples' ჩაანაცვლებს 'milk'-ს, ხოლო დანარჩენი ორი (bread და eggs) უცვლელი დარჩება.


(სანამ ES6 ვერსიაში (თანამედროვე JavaScript) Default Parameters გამოჩნდებოდა, პროგრამისტები იგივე შედეგისთვის ხშირად იყენებდნენ || (OR) ოპერატორს.
   მაგ. 
function makeShoppingList(item1) {
  item1 = item1 || 'milk';
  console.log(item1);
}  // ლოგიკა: თუ item1–ს  არაფერი გადავეცით, || ოპერატორი აირჩევს მეორე, true მნიშვნელობას — 'milk'.)

დავალება:

შექმენი BMI (სხეულის მასის ინდექსის) კალკულატორი JavaScript-ის ფუნქციების გამოყენებით.
BMI არის სხეულის ცხიმის რაოდენობის გამოთვლის საშუალება, 
რომელსაც მედიცინაში გულ-სისხლძარღვთა დაავადებების რისკის შესაფასებლად იყენებენ.

ფორმულა:BMI გამოითვლება შემდეგნაირად: წონა გაყოფილი სიმაღლის კვადრატზე.
BMI ={weight} / {height^2}

შენი დავალება:შექმენი ფუნქცია, რომელიც შეყვანილ წონასა და სიმაღლეს გამოიყენებს BMI-ის გამოსათვლელად.
ფუნქციამ არგუმენტებად უნდა მიიღოს ორი მნიშვნელობა: პირველი — წონა (weight), მეორე — სიმაღლე (height).
ფუნქციამ უნდა დააბრუნოს (return) მიღებული BMI-ის მნიშვნელობა.
შედეგი უნდა იყოს დამრგვალებული უახლოეს მთელ რიცხვამდე (გამოიყენე Math.round()).
მნიშვნელოვანი შენიშვნა: * ამ დავალებაში არ დაგჭირდება alert(), prompt() ან console.log(). კოდი უნდა შეიცავდეს მხოლოდ ფუნქციას.
ფუნქციამ შედეგი უნდა დააბრუნოს (return) და არა უბრალოდ დაბეჭდოს.
